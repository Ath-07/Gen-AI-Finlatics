# Generative AI Systems

Production-oriented **Generative AI applications** demonstrating hands-on experience with **Large Language Models, NLP, prompt engineering, and real-time conversational systems**.  
The projects focus on **LLM integration, controllable generation, and system-level design considerations**.

---

## Overview

This repository contains two end-to-end Generative AI systems designed to reflect **real-world LLM-powered applications**, including multilingual translation and controlled conversational generation.

Key focus areas:
- LLM-driven conversational systems  
- Prompt engineering and response control  
- Token usage monitoring and session analytics  
- Streaming output for low-latency user experience  

---

## System 1: Multilingual Translation Chatbot

A session-aware chatbot enabling **English to French translation** using a pre-trained neural machine translation model.

### Capabilities
- Real-time translation using Hugging Face NLP pipelines  
- Word-level streaming responses for improved interactivity  
- Conversation memory with timestamped history  
- Input and output token tracking for usage analysis  
- Command-driven conversational workflow  

### Tech Stack
- Python  
- Hugging Face Transformers  
- Neural Machine Translation models  

---

## System 2: Controlled-Generation Chatbot with Gemini

A configurable conversational agent built on **Google Gemini**, supporting creative and factual response modes through structured prompt design.

### Capabilities
- Gemini API integration for real-time text generation  
- Prompt-driven dual-output creative and factual responses  
- Mood-conditioned response generation  
- Session-level conversation summarization  
- Token usage estimation and monitoring  

### Tech Stack
- Python  
- Google Gemini API  
- Prompt Engineering  
- Large Language Models
